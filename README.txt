# 🏆 Clicker to Get Big Trophies

Welcome to **Clicker to Get Big Trophies**!

This is a clicker game where you earn points, unlock trophies, progress through unique prestige levels, and discover exclusive trophy series inspired by gaming memories and personal experiences.

## 🎮 Features

- 🏆 Hundreds of trophies to unlock
- 👑 Unique Prestige System
- 🌿 Memory Trails
- 🤠 Frontier Memories
- 🎁 Unique Perks with special bonuses
- 🎨 Unlockable Themes
- 💾 Automatic Save
- 📱 Installable as a Progressive Web App (PWA)

## 🌍 Language

Please note that **most of the game is currently written in Swedish**, as I am a Swedish-speaking developer.

If you don't speak Swedish, you can easily translate the game using **Google Chrome's built-in translation feature**, which works well for most of the interface.

A **language selector is planned for a future update**, allowing players to switch between Swedish and English directly inside the game.
